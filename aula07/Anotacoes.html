Tipos de Operadores:

aritméticos, atribuição, relacionais, lógicos, ternarios

Operadores aritméticos:

+ --> Somar

- --> Subtrair

* --> Multiplicar

/ --> Divisão

% --> Resto da divisão

** --> Potência

Ordem de precedência:

()

**

/ * %

+ -

Operadores de atribuição:

n = 3

n = n + 4 //vai somar n(3) + 4 = 7

n = n - 2 //vai subtrair n(3) - 2 = 1

n = n * 4 //vai multiplicar n(3) * 4 = 12

n = n / 6 //vai dividir n(3) / 6 = 2

n = n ** 2 //vai elevar a 2° potência = 9

n = n % 5 //vai dividir n(3) / 5 = 2 //resto da divisão

Forma Simplificada:

n += 4

n -= 2

n *= 4

n /= 6

n **= 2

n %= 5

Incremento:

n++ //vai incrementar + 1 na variavel

n-- //vai diminuir - 1 na variavel

++n //vai incrementar antes

--n //vai diminuir antes

Operadores relacionais:

5 > 2 -> true

7 < 4 -> false

8 >= 8 -> true

9 <= 7 -> false

5 == 5 -> true

4 != 4 -> false

Exemplos:

preco >= 200.50 //o preço é maior ou igual a 200.50?

idade < 18

//a idade é menor que 18 anos?

curso == 'JavaScript' //o curso é igual a JavaScript?

n1 != n2 //o primeiro numero é diferente do segundo?

Identidade:

5 == 5 -> true //5 é igual a 5? Sim

5 == '5' -> true //5 é igual a string 5? Sim

5 === '5' -> //5 é indêntico a string 5? Não

Operadores Lógicos:

! -> Negação

&& -> Conjunção

|| -> Disjunção

Negação:

Uma coisa que é verdadeira é falsa

Uma coisa que é falsa é verdadeira

Conjunção:

Se for true && true é true

Se for true && false é false

Se for false && false é true

Se for false && true é false

Disjunção:

Se for true ou false é true

Se for false ou false é false

Se for false ou true é true

Se for true ou true é true

Exemplos:

idade >= 15 && idade <= 17 //a idade está entre 15 e 17?

estado == 'SP' || estado == 'RJ' //o estado é SP ou RJ?

salario > 1500 && sexo != 'M' //o salario pe acima de 1500 e não é um homem?
