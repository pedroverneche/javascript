VARIÁVEIS:

Como criar variáveis? ex: var nome ou let nome

REGRAS DAS VARIÁVEIS:

Podem começar com letra, '$' ou '_'

Não podem começar com números

É possível misturar letras com números

Pode usar acentos e símbolos

Não pode conter espaços

Não pode usar palavras que são comandos

DICAS PARA NOMES DE VARIÁVEIS:

Maiúsculas e Minúsculas fazem a diferença!

Tente escolher nomes coerentes para variável. ex: A variável que armazena idade recebe o nome "idade"

Evite se tornar um “Programador Alfabeto” ou um “Programador Numérico”.

Ex: Não usar os nomes das variáveis como “a”, “b”, “c”, etc...

ou “a1”, “a2”, “a3”, etc...

Tipos de Dados das Variáveis(Tipos Primitivos):

Numbers, Strings, Boolean;

Numbers: 1, 4, -6, 7.9, 6.113 -> Basicamente qualquer tipo de número

Strings: Pedro, programador, Itanháem -> cadeia de caracteres

Boolean: True, False -> Tipos lógicos
