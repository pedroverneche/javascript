Conversores de String para Number:

Number.parseInt(n) -> converte a string em um número inteiro

Number.parseFloat(n) -> converte a string em um número real

Number(n) -> converte a string em um número inteiro e real(RECOMENDADO)

EX: var n1 = Number(window.prompt('Digite um número: '))

Conversores de Number para String:

Ex: window.alert ('a soma dos numeros é: ' + soma.toString()) // Jeito mais antigo

Ex: window.alert ('a soma dos numeros é: ' + String(soma)) // Jeito mais simples

Formatando Stirngs:

Formatando Números:

var n1 = 1543.5

n1.toFixed(2) // Coloca em duas casas decimais (para colocar mais ou menos casas troque o numero entre parênteses)

n1.toLocaleString('pt-BR',{style: 'currency', currency: 'BRL'}) // mostra o R$ na frente do numero (pode trocar entre outras moedas)
